title: How to Save and Print as PDF in valuationIQ for Mac
---

1. To Save or Print a file as a PDF in Excel for Mac, first you will need to click the "Show Model" button on the ribbon to unhide all tabs.

<Frame>
  <img src="/images/rediq/faa93b5a54fb9ec173661add0de41290df3421c1.png" alt="Show Model Excel for Mac.png" />
</Frame>

2. Hold down the Shift key and select "OUTPUT TABS ==>" then "Sources & Uses" so all Output tabs are selected.

<Frame>
  <img src="/images/rediq/874af0c9734868fceff5b7f2e3bfaecca955f802.png" alt="Output Tabs Excel for Mac.png" />
</Frame>

3. Navigate to File -> Print, or press Command + P. 

4. In the print screen, select the "Margins" dropdown then the "Manage Custom Margins" option.

![Manage Custom Margins Excel for Mac.png](/images/rediq/f7438359c7d6c66fb351a8638b9d739fde0cfd4d.png)5. For scaling, the fit should be 1 pages wide by 1 page tall. Click ok.

![Scaling Excel for Mac.png](/images/rediq/d1dd6d85c382b6a31deeeef59a676b8c8c91b156.png)6. Select the "PDF" dropdown and "Save as PDF". You can then name and save your file to your computer.

  <img src="/images/rediq/b65d69ad07a60425723d361f4b56fb0271d89627.png" alt="Save as PDF Excel for Mac.png" />
</Frame>
