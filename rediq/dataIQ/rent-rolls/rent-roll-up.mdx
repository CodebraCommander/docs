title: Rent Roll Up
---

The rent roll up feature allows users to combine rent rolls already in redIQ into one consolidated rent roll. This will help users who are looking to consolidate multiple properties or phases into a single deal.


 In the **Comps** screen, select the **Rent Roll**tab. You can select the deals you would like to Roll Up.


<Frame caption="Comps section displaying selected rent roll deals with filtering options.">
  <img src="/images/rediq/f710e65e94fbfd22aa9155767c68637cac197b0a.jpg" alt="rent_roll_up_ppic_1.jpg" />
</Frame>


Select the **Roll Up**button on the right-hand side. Then select **Rent Roll** in the drop-down menu.


<Frame caption="Dropdown menu for selecting report types: Rent Roll, Unit Mix, Dashboard.">
  <img src="/images/rediq/d1b2c5c12822912d62dee40e7b280c0003cf7857.png" alt="rent_roll_up_ic_2.png" />
</Frame>


A window will appear prompting the user to select the desired As of Date


<Frame caption="Select rent rolls from deals for the roll-up process.">
  <img src="/images/rediq/66e3985a1b023768df0f88f629233ba45ea0a7e2.png" alt="pic123455.png" />
</Frame>


The consolidated Rent Roll will now appear. Select the Export to Excel button in the top right.


<Frame caption="Rent Roll Roll Up interface displaying unit details and report settings.">
  <img src="/images/rediq/b7583a2e459b1260e6b283dc62ed65d3519b3edc.png" alt="pic1234566.png" />
</Frame>


Once the consolidated rent roll is exported, you may save the file, and upload into a new deal.
